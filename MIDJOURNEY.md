# Midjourney prompts — HTPC upgrade posts

One prompt per fenced block (the `midjourney-submit` skill reads it that way). Filter by section
heading, e.g. `hero`, `card`, `transmission`, `flaresolverr`, `ombi`.

- **Hero** = `bigimg` banner, `--ar 4:1` (the theme crops a center strip to ~5.3:1).
- **Card** = `img` thumbnail, `--ar 5:4`.
- Brand-/text-free on purpose: the theme overlays its own title, and Midjourney mangles logos.
- On `--v 7` — drop to your subscription's model if needed.
