# Midjourney — post thumbnails

Replacements for the recycled `Home Media Center-Small.jpg` used across the
Home-Media-Server series. On brand with the **Dark Sea-Chart** theme:
deep navy + brass + verdigris-teal, antique nautical-chart engraving, the
"*arr = arrr, torrents = the high seas" media-automation dream.

## Shared style suffix

Append to every prompt for a consistent set (v7). Seed `--sref` from `logo.png`
once you have a URL you like, then reuse the same value everywhere so the whole
series matches.

```
antique nautical sea-chart engraving, deep navy background, brass and verdigris-teal accents, weathered parchment texture, fine copperplate linework, subtle compass rose and rhumb lines, cinematic low-key lighting, no text, no lettering --ar 3:2 --style raw --stylize 250
```

Thumbnails render small, so favor **one strong silhouette** over busy detail.
